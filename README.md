## Getting Started

### Prerequisites

You need to install nodejs before installing

npm

```sh
npm install npm@latest -g
```

### Installation

Clone the repo

```sh
git clone https://github.com/KeyQorLLC/ExchangePcs.git
```

Install NPM packages

```sh
cd frontend
npm install
```

```sh
cd backend
npm install
```
